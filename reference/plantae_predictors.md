# Predictor variable names for plantae dataset

Character vector of 84 predictor variable names from `plantae`, covering
sampling bias (1), geographic/geometric (10), human impact (3), climate
change (2), landcover (3), fragmentation (19), atmospheric (13),
precipitation (8), temperature (11), soil properties (6), soil
temperature (4), and NDVI (4).

## Usage

``` r
data(plantae_predictors)
```

## Format

A character vector of length 84.
