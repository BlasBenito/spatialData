# Response variable names for plantae dataset

Character vector containing the names of the 53 response variables in
`plantae`, covering richness metrics (9), rarity-weighted richness (6),
mean rarity (6), and beta diversity indices including raw turnover (16),
Sorensen (8), and Simpson (8) dissimilarities. Metrics are available for
all plants, trees, and grasses at species, genus, and family levels.

## Usage

``` r
data(plantae_responses)
```

## Format

A character vector of length 53.
