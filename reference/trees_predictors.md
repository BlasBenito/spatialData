# Predictor variable names for trees dataset

Character vector of 50 predictor variable names from
[trees](https://blasbenito.github.io/spatialData/reference/trees.md),
covering air humidity (4), aridity (1), cloud cover (4),
evapotranspiration (4), precipitation (8), temperature (11), geography
(4), soil properties (6), soil temperature (4), and NDVI (4).

## Usage

``` r
data(trees_predictors)
```

## Format

A character vector of length 50.
