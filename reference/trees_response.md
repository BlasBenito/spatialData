# Response variable name for trees dataset

Character vector of length 1 containing the name of the response
variable in
[trees](https://blasbenito.github.io/spatialData/reference/trees.md):
`"trees"` (integer count of tree species richness per hexagonal cell).

## Usage

``` r
data(trees_response)
```

## Format

A character vector of length 1.
